# 云控制台最小权限模型

## 原则
- 每个服务账号仅授予完成其职责所需的最小权限
- 定期审计权限使用情况

## 权限矩阵

| 资源 | Gateway 节点 | OS 节点 | CI/CD | 运维人员 |
|------|-------------|---------|-------|---------|
| 对象存储 | 无 | 读写（备份） | 写（产物） | 读 |
| 镜像仓库 | 拉取 | 拉取 | 推送 | 读 |
| 数据库 | 无 | 读写 | 无 | 只读 |
| 密钥管理 | 读取自身密钥 | 读取+签发 | 无 | 轮换 |
| 日志服务 | 写 | 写 | 无 | 读 |
