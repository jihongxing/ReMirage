# 错误处理

## 错误码

| 代码 | 名称 | 说明 | 处理方式 |
|------|------|------|---------|
| 0 | OK | 成功 | - |
| 1 | UNAUTHENTICATED | 认证失败 | 刷新 Token |
| 2 | QUOTA_EXCEEDED | 配额不足 | 购买流量包 |
| 3 | CELL_UNAVAILABLE | 蜂窝不可用 | 切换蜂窝 |
| 4 | INVALID_ARGUMENT | 参数错误 | 检查请求参数 |
| 5 | INTERNAL | 内部错误 | 重试或联系支持 |
| 6 | RATE_LIMITED | 请求过频 | 降低请求频率 |
| 7 | PERMISSION_DENIED | 权限不足 | 检查 Token scope |

## gRPC 错误处理

### Python

```python
import grpc
from mirage import MirageClient

client = MirageClient(endpoint, token)

try:
    response = client.gateway.sync_heartbeat(request)
except grpc.RpcError as e:
    if e.code() == grpc.StatusCode.UNAUTHENTICATED:
        # 刷新 Token
        pass
    elif e.code() == grpc.StatusCode.RESOURCE_EXHAUSTED:
        # 配额不足
        pass
    elif e.code() == grpc.StatusCode.UNAVAILABLE:
        # 服务不可用，重试
        pass
    else:
        raise
```

### Go

```go
resp, err := client.Gateway.SyncHeartbeat(ctx, req)
if err != nil {
    st, ok := status.FromError(err)
    if ok {
        switch st.Code() {
        case codes.Unauthenticated:
            // 刷新 Token
        case codes.ResourceExhausted:
            // 配额不足
        case codes.Unavailable:
            // 重试
        }
    }
    return err
}
```

### JavaScript

```typescript
try {
  const response = await client.gateway.syncHeartbeat(request);
} catch (error) {
  if (error.code === grpc.status.UNAUTHENTICATED) {
    // 刷新 Token
  } else if (error.code === grpc.status.RESOURCE_EXHAUSTED) {
    // 配额不足
  }
}
```

## 重试策略

### 指数退避

```python
import time
import random

def retry_with_backoff(func, max_retries=3):
    for attempt in range(max_retries):
        try:
            return func()
        except grpc.RpcError as e:
            if e.code() not in [grpc.StatusCode.UNAVAILABLE, grpc.StatusCode.DEADLINE_EXCEEDED]:
                raise
            if attempt == max_retries - 1:
                raise
            delay = (2 ** attempt) + random.uniform(0, 1)
            time.sleep(delay)
```

### 可重试错误

| 错误 | 可重试 | 建议延迟 |
|------|--------|---------|
| UNAVAILABLE | ✅ | 1-5s |
| DEADLINE_EXCEEDED | ✅ | 1-5s |
| RESOURCE_EXHAUSTED | ⚠️ | 30s+ |
| INTERNAL | ⚠️ | 5-10s |
| UNAUTHENTICATED | ❌ | - |
| INVALID_ARGUMENT | ❌ | - |

## WebSocket 错误

### 断线重连

```javascript
class ReconnectingWebSocket {
  constructor(url, token) {
    this.url = url;
    this.token = token;
    this.reconnectDelay = 1000;
    this.maxDelay = 30000;
  }

  connect() {
    this.ws = new WebSocket(this.url);
    
    this.ws.onclose = () => {
      setTimeout(() => {
        this.reconnectDelay = Math.min(this.reconnectDelay * 2, this.maxDelay);
        this.connect();
      }, this.reconnectDelay);
    };

    this.ws.onopen = () => {
      this.reconnectDelay = 1000;  // 重置
      this.ws.send(JSON.stringify({ type: 'auth', token: this.token }));
    };
  }
}
```
