# Справочник API

## Обзор сервисов

| Сервис | Порт | Протокол | Назначение |
|--------|------|----------|------------|
| Gateway | 50847 | gRPC | Heartbeat/Трафик/Угрозы |
| Cell | 50847 | gRPC | Управление ячейками |
| Billing | 50847 | gRPC | Биллинг/Пополнение |
| WebSocket | 18443 | WSS | Push в реальном времени |

---

## GatewayService

### SyncHeartbeat - Синхронизация Heartbeat

**Параметры запроса**

| Поле | Тип | Обязательно | Описание |
|------|-----|-------------|----------|
| gateway_id | string | ✅ | Уникальный идентификатор Gateway |
| version | string | ✅ | Версия Gateway |
| threat_level | uint32 | ❌ | Текущий уровень угрозы (0-5) |

**Параметры ответа**

| Поле | Тип | Описание |
|------|-----|----------|
| success | bool | Статус успеха |
| remaining_quota | uint64 | Оставшаяся квота (байты) |
| next_heartbeat_interval | int64 | Интервал следующего heartbeat (секунды) |

---

### ReportThreat - Отчет об угрозах

**Типы угроз (ThreatType)**

| Значение | Описание |
|----------|----------|
| ACTIVE_PROBING | Активное зондирование |
| JA4_SCAN | Сканирование отпечатка JA4 |
| SNI_PROBE | SNI зондирование |
| DPI_INSPECTION | Глубокая инспекция DPI |
| TIMING_ATTACK | Атака по времени |
| REPLAY_ATTACK | Атака воспроизведения |
