# Source of Truth Record Template

## Problem Domain

说明该真相源负责哪一类事实。

## Authoritative Source

填写唯一主真相源的位置。

## Why This Source Owns It

解释为什么由这里持有主权，而不是其他文档或代码。

## Allowed Derived Artifacts

列出哪些派生物可以存在，例如教程、样例、摘要、生成代码。

## Explicit Non-Sources

列出容易被误认成真相源，但实际上不是的材料。

## Update Workflow

描述这类事实发生变更时，应该先改哪里，再同步哪里。
