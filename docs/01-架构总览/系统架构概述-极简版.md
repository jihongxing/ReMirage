# Mirage Project 系统架构概述（极简版）

## 项目定位

高级流量隐匿与反审计系统，为高净值用户提供"感知-防御-逃逸"的自动化闭环。通过内核态拟态化与基础设施碎片化，在物理空间、地理政治和统计学维度同时实现消失。

---

## 核心组件

### Mirage-Gateway（融合网关）

**职责**：边缘感知 + 流量隐匿传输

**核心能力**：
- 流量指纹识别（JA3/JA4 黑名单 + 异常检测）
- 诱饵流检测（识别扫描器）
- eBPF 透明拦截（零侵入流量接管）
- 零拷贝性能优化（< 3% CPU 开销）
- 动态 TLS/QUIC 栈指纹伪装
- 时域扰动（Jitter-Lite 微秒级 IAT 随机化）

### Mirage-OS（控制中心）

**职责**：全球大脑 + 威胁情报聚合

**核心能力**：
- 用户蜂窝管理（200 用户分组隔离）
- 威胁情报聚合（全局黑名单同步）
- 集群自治（自膨胀、热迁移、死信开关）
- 资产保护（全内存运行、盲备份）
- 生死裁决（配额熔断、流量结算）
- 实时推送（WebSocket + Redis Pub/Sub）

## 核心技术特性

### 1. eBPF 内核加速

- XDP/TC 层透明拦截
- 零拷贝数据传输
- 性能开销 < 3%
- 支持 Kernel ≥ 5.15（Fallback ≥ 4.19）

### 2. G-Tunnel 多路径传输

- AVX-512 加速 FEC 编码（< 1ms）
- 8 数据分片 + 4 冗余分片
- 33% 丢包容忍率
- 转生协议（5 秒路径切换）

### 3. 全球拟态矩阵

- 6 种流量模板（Zoom/Netflix/WhatsApp/Steam/Twitch/Skype）
- 自动识别业务类型
- 动态切换拟态指纹
- 区域化流量特征库

### 4. 司法管辖区轮转

- 规避 14 眼情报联盟
- 三跳隔离原则
- 自动评分系统
- 匿名采购流程（Monero 混币）

### 5. M.C.C. 绝对物理隐匿

- 四层防护（逻辑脱钩 + 地址瞬态化 + 流量掩护 + 物理隔离）
- 全内存无盘运行（tmpfs 部署）
- Shamir 碎片化存储（3-of-5 门限）
- 死信开关（300 秒心跳超时自毁）
- RAM Shield（mlock 内存锁定）
- 紧急自毁（eBPF Map 原子清空）

### 6. 实时态势感知

- 3D 地球可视化（Three.js）
- WebSocket 实时推送
- 威胁情报流
- 配额消耗监控
- 拟态相似度实时曲线
- 全球拦截热图
- 转生记录时间线

### 7. 自动化红蓝对抗

- AI 审计评估器（流量特征分析）
- 闭环反馈控制（自动参数调整）
- 异常分数实时监控（< 20% 阈值）
- 配置文件动态切换
- 持续自我进化

- 心跳丢失 > 5min → 自动清零

### 匿名性保障
- Monero 混币支付
- 一次性身份信息
- 隐私注册商（Njalla/1984）
- Tor Hidden Service 备用通道

## 部署架构

### M.C.C. 集群拓扑（Raft 一致性）
- 节点 #1: 冰岛（IS）- 数据隐私保护
- 节点 #2: 瑞士（CH）- 银行保密法
- 节点 #3: 新加坡（SG）- 中立司法管辖区
- 节点 #4: 巴拿马（PA）- 冷备
- 节点 #5: 塞舌尔（SC）- 冷备

### Raft 一致性
- Quorum: 3-of-5
- 自动故障转移
- 状态实时同步
- 威胁感知退位（Leader 自动让位）
- 地理围栏监控

### 全球蜂窝网络（影子蜂窝）
- 三阶段生命周期：潜伏 → 校准 → 服役
- 潜伏期：VPC 噪声注入（1 小时）
- 校准期：网络质量测量（100 次采样）
- 服役期：真实流量承载
- 动态伸缩（威胁等级 ≥ 3 或负载 ≥ 80%）

## 核心优势

1. **时空双重消失**：时域扰动 + 空间拟态
2. **全球免疫系统**：单点受袭，全网封禁（60 秒）
3. **物理层隐匿**：M.C.C. 逻辑幻影 + 物理脱钩
4. **极限容错**：30% 丢包下 0 丢失
5. **成本透明**：按量计费 + 实时熔断
6. **自进化能力**：威胁情报自动学习

## 战略定位

Mirage Project 不是简单的 VPN 或 Proxy，而是一个自进化、自计费、自防御的去中心化网络协议栈。它不属于任何一个国家，也不受限于任何单一视角。

**核心理念**：最好的隐藏，是让自己变成背景。
