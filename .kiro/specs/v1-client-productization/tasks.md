# Implementation Plan: Client 产品化（拓扑学习 + 订阅托管 + 后台服务化）

## Overview

将 phantom-client 从前台 CLI 原型升级为可长期常驻、自愈、可托管的正式产品形态。按依赖顺序实施：基础设施层 → 拓扑层 → 订阅层 → 服务层 → 主入口重构 → 集成联调。
